# 🌌 SDSkyblockCore

A powerful Minecraft server plugin inspired by Hypixel Skyblock. Bring the ultimate Skyblock experience to your players with custom items, unique abilities, player stats, and persistent database storage.

## 🛠️ Requirements

To ensure the plugin runs smoothly, make sure your server meets the following specifications:

*   **Java Version:** Java 21
*   **Minecraft Version:** 1.21.1
*   **Server Software:** [Paper](https://papermc.io/) *(Recommended and fully tested)*

> ⚠️ **Note:** This plugin is strictly built and tested for **Minecraft 1.21.1**. Compatibility with other versions is not guaranteed.

---

## ✨ Features

*   **Custom Items & Abilities:** Recreate the iconic Hypixel Skyblock feel with custom weapons, tools, and item abilities.
*   **Player Stats System:** Tracks and manages player attributes seamlessly.
*   **Database Integration:** Saves and loads player data automatically on join and leave to prevent data loss.

---

## 💻 Commands & Permissions

| Command | Description | Example |
| :--- | :--- | :--- |
| `/sbget <item_id>` | Give yourself a specific custom Skyblock item. | `/sbget aspect_of_the_end` |
| `/sditems` | Open the custom items menu/management interface. | `/sditems` |
| `/sdskills <add\|remove> <skill_type> <amount>` | Modify player skill levels manually. | `/sdskills add combat 5` |

---

## 🚀 Getting Started

1. Download the latest `.jar` file from the [Releases](https://github.com/sumit282698/SDSkyblockCore/releases) tab.
2. Drop the jar file into your server's `plugins/` directory.
3. Restart your server.
4. Configure your database settings in the generated `config.yml` (if applicable).
